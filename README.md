🛡️ HYPER SCANNER ULTIMATE
Разработчик: ksipx
Статус: Private / Release

HYPER SCANNER — это ультимативный, портативный швейцарский нож (SS-tool) для модераторов и администраторов Minecraft серверов. Программа создана для максимально быстрого, глубокого и удобного поиска читов, скрытых модификаций и следов использования стороннего ПО на компьютере подозреваемого игрока.

Вам больше не нужно открывать десятки окон и вводить консольные команды вручную — весь необходимый арсенал собран в одном кибер-интерфейсе.

⚡ Ключевые возможности
Глубокий анализ системы (Deep Scan): Автоматический поиск скрытых следов читов (Vape, Expensive, NurSultan и др.) в системных директориях Windows (Prefetch, Temp).

Мониторинг процессов в памяти: Мгновенное сканирование всех запущенных процессов на наличие известных инжекторов и чит-клиентов.

Анализ кэша DNS: Проверка истории сетевых запросов. Позволяет узнать, заходил ли игрок на сайты по продаже читов (intent.store, vape.gg) или скачивал ли подозрительные конфиги.

Умный проводник Minecraft: Автоматически находит папки .minecraft и TLauncher. Показывает содержимое папок mods и versions.

Киллер-фича: Программа автоматически вычисляет и показывает вес .jar файлов внутри папок с версиями, позволяя моментально вычислить подмену клиента.

Глубокий поиск по логам: Инструмент ищет заданный текст (например, название чита, макроса или никнейм) внутри всех файлов .log и .txt.

Галерея улик: Встроенный просмотрщик последних сделанных скриншотов из игры.

База Regex-запросов: Готовые, сложные регулярные выражения для поиска читов и конфигов через утилиту Everything (копируются в один клик).

🛠️ Встроенный арсенал (Portable)
Программа поставляется в виде одного .exe файла, внутри которого уже зашиты лучшие утилиты для проверок (не требуют установки):

Everything — для моментального поиска файлов по всему диску.

LastActivityView — для просмотра истории запуска любых программ и открытия папок.

ShellBag Analyzer — для поиска следов удаленных папок с читами.

Recuva — для восстановления "случайно" удаленных файлов прямо во время проверки.

💎 Преимущества
Полная портативность: Один файл. Скинул подозреваемому, он запустил — вы проверяете. Никаких следов после закрытия.

Мультиязычность: Поддержка русского и английского языков (выбор при старте).

Premium UI: Инновационный дизайн в стиле Cyberpunk/Neon с плавными анимациями, космосом и защитой от зависаний.
